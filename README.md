# WanderTrails

WanderTrails is a React-based web application for discovering outdoor tours and adventures.
Users can browse available tours, view details, and explore destinations in a clean and modern UI.

## Features
- Browse outdoor tours
- View detailed tour information
- User authentication (mock)
- Protected routes
- Responsive design

## Tech Stack
- React
- React Router
- Context API
- CSS
- Create React App

## Getting Started

## To run the project locally:
```bash
npm install
npm start
```

The app will be available at:
http://localhost:3001

## Project Purpose

This project was created as a portfolio project to demonstrate:
- React fundamentals
- Component-based architecture
- State management with Context API
- Routing and protected pages
